## Global Design System

### Color Palette
**From existing site + expansion:**
- **Primary Dark:** #161317 (text, buttons, dark sections)
- **Pure Black:** #000000 (headings)
- **Pure White:** #ffffff (backgrounds, light text)
- **Gray Accents:** #d0d0d0, #dedede (borders, subtle elements)
- **Hover State:** #3e3641 (button hover, interactive elements)
- **Light Background:** #f9f9f9 (section alternation)

### Typography System
**Headings:** Marcellus (serif)
- H1: 60-80px, bold
- H2: 40-50px, bold
- H3: 30-36px, bold

**Body:** DM Sans (sans-serif)
- Body Text: 18-21px, regular
- Small Text: 14-16px, regular
- Button Text: 16-18px, medium weight

**Base Size:** 21px (from existing site)

### Spacing System
- **Section Padding:** 80-100px top/bottom (desktop), 60px (mobile)
- **Container Max-Width:** 1200px (wide), 1000px (standard), 800px (narrow)
- **Grid Gaps:** 30-40px between cards/columns

### Button Styles
**Primary:**
- Background: #161317
- Text: #ffffff
- Padding: 16px 40px
- Border-radius: 4px (slight) or 0px (sharp)
- Hover: #3e3641 background

**Secondary (Ghost):**
- Background: transparent
- Border: 2px solid #161317
- Text: #161317
- Hover: Fill with #161317, text white

### Image Treatment
- **Style:** B&W photography (existing aesthetic)
- **Aspect Ratios:**
  - Hero: 16:9 or full screen
  - About photos: 3:4 portrait
  - Testimonial: 1:1 circle
- **Effects:** Subtle shadows, no heavy filters

---

## Responsive Breakpoints

```
Mobile:    < 768px  (stack all columns, full-width elements)
Tablet:    768-1024px (2-column layouts, adjusted spacing)
Desktop:   > 1024px (full multi-column layouts, max spacing)
```

**Mobile Considerations:**
- Stack all multi-column sections
- Reduce font sizes by 20-30%
- Full-width CTAs
- Reduce section padding to 60px
- Hero section 70vh minimum
