# Shannon Muruli Brand Website

A professional, conversion-optimized website for Shannon Muruli - Courage Coach & Self-Image Strategist to Women Who Sell.

## Project Overview

This website reimagines theuntetheredweekly.com as a multi-page personal brand website inspired by the structure and polish of marieforleo.com, while maintaining Shannon's authentic "Courage Over Comfort" brand voice and "The Untethered Seller" identity framework.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Typography:** Marcellus (headings), DM Sans (body)
- **Language:** TypeScript
- **Deployment Ready:** Vercel

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production
```bash
npm run build
npm start
```

## Pages

1. **Homepage** (/) - 8-section conversion funnel
2. **About** (/about) - Full origin story with photos
3. **Work With Me** (/work-with-me) - 3 coaching programs + FAQ
4. **Client Results** (/client-results) - Testimonials and transformations
5. **Contact** (/contact) - Contact form + newsletter CTA
